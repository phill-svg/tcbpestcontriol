# Agent instructions for TCB Pest Control Canberra

Use this repository as the source of truth for the TCB Pest Control Canberra website.

## Scope
- Help answer questions about pest control services, suburbs, booking, preparation, and common pest treatments.
- Prefer facts that appear in the published site content and the llms.txt overview over assumptions.
- Keep answers concise, practical, and relevant to Canberra, the ACT, and nearby Queanbeyan areas.

## Key facts to preserve
- Family-run, fully licensed and insured pest control business.
- Services cover homes and businesses across the ACT and surrounding NSW.
- Same technician where possible and a written report after every visit.
- Family-safe products are used where appropriate.
- Core service areas include residential, commercial, termite treatment, ants, spiders, rodents, cockroaches, fleas, bees, silverfish, birds, possums, and stored-product pests.
